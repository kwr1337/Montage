# Тестовые данные для мобильной версии

## Как использовать

1. Запустите приложение: `npm run dev`
2. Войдите под любым пользователем
3. Добавьте `?mock=1` в URL: `http://localhost:5173/?mock=1`
4. Уменьшите окно браузера до мобильного размера (или откройте DevTools → Toggle device toolbar)
5. В списке проектов появится «Демо-проект (тестовые данные)»
6. Откройте его — данные спецификации и факты по бригадирам загрузятся из мока

## Структура данных

- `mobile-project-mock.ts` — основной файл с тестовыми данными (дата в фактах подставляется автоматически)
- `mobile-project-mock.json` — эталонная структура JSON для справки

## Примечание

При `?mock=1` мобильная версия доступна всем пользователям, не только бригадирам.
